---

## 🚀 Features

* **Two-Player Mode:** Play turn-by-turn with a friend on the same device.
* **Win & Draw Detection:** Automatically checks for horizontal, vertical, diagonal wins, or a full-board tie.
* **Simple UI:** Clean interface built using Java Swing (or CLI).
* **Reset Option:** Easily start a new game after a match finishes.

---

## 🛠️ Requirements

* **Java Development Kit (JDK):** Version 8 or higher
* Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or command line

---

## 🕹️ How to Play

1. Players take turns clicking (or selecting) an empty square on the **3x3 grid**.
2. The first player plays as **X**, and the second player plays as **O**.
3. The first player to align **3 of their marks horizontally, vertically, or diagonally** wins the game.
4. If all 9 squares are filled and no player has 3 in a row, the game ends in a **Draw**.
